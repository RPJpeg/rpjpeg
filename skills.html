WIREFRAMING

PLANNING

CODING

I start out all my projects by wireframing the general idea in Adobe Illustrator, Adobe XD or Sketch. At this point I'm just trying to figure out how the structure and flow of the site is going to work. I decide how many pages will be included and where each sectino will be placed.

Next, I will start the planning stage. I take the prototype that I created beforehand and I start to dissect it. I'll break it up into sections, divs, list and typography elements. I select a color palette and estimate the size of each element.

Finally, I'll take my plan and start to convert it to code, using a combination of HTML, CSS and Javascript. My text editor of choice is Atom, and I prfer using optimization tools such as SASS and jQuery to streamline my process to make sure the code created is minified and super clean. In some cases I will use CSS/Javascript frameworks for assistance, such as Twitter Bootstrap or Bourbon.
